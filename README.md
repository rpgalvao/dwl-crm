# DWL CRM - Custom Sales Pipeline

A modern, tailored Customer Relationship Management (CRM) system developed specifically for Diagnostic Work Logic (DWL). This MVP streamlines the sales process, from lead generation to deal closure, providing a clear executive overview of the company's performance.

## 🚀 Key Features

- **Interactive Sales Funnel:** Drag-and-drop Kanban board to manage deals across different stages (New, Quoting, Pending Approval, Won).
- **Product/Reagent Catalog:** Full CRUD management for DWL's product line with automated price filling.
- **Contact Management:** Centralized client database with automatic Brazilian phone formatting and foreign key protection.
- **Executive Dashboard:** Real-time metrics, financial summaries, and a visual Doughnut chart for deal distribution.
- **Brand Identity:** Custom UI/UX strictly following DWL's Brand Board (colors, typography, and logo).

## 🛠️ Tech Stack

- **Backend:** Laravel 11, PHP
- **Frontend:** Vue.js 3, Inertia.js, Tailwind CSS
- **Database:** PostgreSQL
- **Charts:** Chart.js & vue-chartjs
